🩺 Vitamin Deficiency Disease Prediction using Machine Learning
📌 Overview

A Machine Learning project that predicts vitamin deficiency-related diseases such as Anemia, Night Blindness, Rickets/Osteomalacia, Scurvy, and Healthy condition based on nutritional, clinical, and lifestyle factors.

🎯 Key Features
Data Preprocessing & Cleaning
Exploratory Data Analysis (EDA)
Feature Selection Comparison
Multiple ML Model Evaluation
Final Random Forest Prediction Model
Deployment using saved model files

🛠️ Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

🏆 Best Results
Best Feature Selection: SelectKBest (Chi-Square)
Best Model: Random Forest Classifier
Accuracy: 93%
F1 Score: 92%

🚀 Outcome

This project helps identify disease categories based on nutrient deficiency markers and supports early prediction through supervised machine learning.
